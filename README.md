# A PHP HTML-to-RTF-converter
_Convert basic HTML output of TinyMCE to RTF with PHP_

Supported HTML:
- Text styles
  - bold
  - italic
  - underline
  - strikethrough
- Align
  - Left
  - Center
  - Right
  - Justify
- Lists
  - Ordered
  - Unordered
- Table
  - tr, th, td
- Unicode characters

Example can be found in examples/index.php
